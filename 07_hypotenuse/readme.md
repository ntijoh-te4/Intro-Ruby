# Hypotenuse

## Beskrivning
Tar två positiva tal som input. Ger som output längden på hypotenusan i en rätvinklig triangel där kateternas längder är input-talen.
### Tips och länkar

Exempel, roten ur samt upphöjt till i Ruby.
````ruby
Math.sqrt(16) #=> 4
2**3 #=> 8
````
